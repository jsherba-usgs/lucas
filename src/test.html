Transition Groups and Types

A set of all possible transitions between land-use types is also defined in the model. The processes represented by these transitions include expansion and contraction of agricultural lands, urbanization, wildfire, shrub encroachment into grassland, harvest of tree plantations and shifts in moisture zones due to climate change.

Select Variable

The LUCAS model tracks land use and carbon through time and space. Depending on the goals of the model and the data available during model development, a range of datasets may be produced. Choosing a dataset to explore further will automatically update the available scenarios and classes.

  • Land-use State: The amount of land-use area in a range of land use classes (e.g. forest, grassland, developed) through time.
  • Land-use Transition: The amoung of land-use area moving from one class to another (e.g. forest -> developed) through time.
  • Carbon Stock: Carbon stocks represent the total carbon stored in different ecosystem pools.

Select Scenarios and Classes

The LUCAS model is typically run over a range of Scenarios. Scenarios represent possible future pathways of change developed with a range of assumptions about land-use management, population dynamics, climate and other variables. Scenarios allow us to ask "what if" questions about how and where change may occur according to an array of differnt futures. Select one or more scenario to explore further.

The Class options will update depending on the chosen variable. Choose one or more classes to further explore.

Select Strata

Choose a spatial unit to summarize results over. Use the dropdown menu or click "Choose on Map" to access a webmap with available spatial layers. I you choose to summarize over more than one strata types results will show data within the overlap area of the strata.

Iterations

The LUCAS model is typically run over multiple Monte Carlo iterations to capture uncertainty within model inputs. Select "Percentile of Iterations" to view the mean and percentile bounds. You may also choose to display results for a single iteration of the model.

Layer controls

Use this interactive web map to explore land use and carbon data accross the landscape. Use the menu below the map frame on the left to update the model layers visible on the web map. These controls allow the user to toggle between different years, scenarios, iterations, and map products. Toggle the "Show map legend" button to show the legend for the current layer on the map. Frame controls

Use the plus and minus buttons on the top left of the web map to add or remove a map frame. When a new map frame is added, layer controls for the frame appear below the frame. Adding a frame allows the user to compare two or more layers side by side.

Base maps

Within each map frame hover over the layers icon to expand the basemap options. Grey-scale and imagery base maps are available as well as strata layers.

Animate map layers

Map layers may be animated through time using the controls below the graph. Choose the layers you would like to animate and then click the play button under the graph. Layers will update every five year timestep within the map frame. The graph tooltip will also animate within the graph frame and show land-use values.
