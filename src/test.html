Transition Groups and Types

A set of all possible transitions between State Types is also defined in the model. For this project, the processes represented by these transitions include expansion and contraction of agricultural lands, urbanization, wildfire, shrub encroachment into grassland, harvest of tree plantations and shifts in moisture zones due to climate change.

The USGS is involved in a variety of scenario-based modeling projects aimed at projecting land-use change into the future. Typically, these projects lead to the creation of many time-series datasets in the form of large tabular and spatial files. A common difficulty among land-use scenario researchers is efficiently managing, analyzing, and visualizing these datasets.

The Geospatial Innovation Facility (GIF) at University of California, Berkeley has developed a prototype web application designed to ingest the tabular and spatial data stored within a generalized USGS land-use database that manages modeling results from ApexRMS's SyncroSim software. The prototype web application consists of the following components:

  • A public Application Programming Interface (API) built on the Django REST framework. The API provides standardized access to data within the SyncroSim database for building web applications and data visualizations.
  • A data management console for researchers to manage projects and scenarios.
  • Interactive web based data visualizations built using the API that allow users to dynamically visualize dense time series of projected climate and land use futures.

This prototype web application allows researchers to explore land-use data scenarios and quickly share land-use modeling results with colleagues and the public. The interactive data visualizations presented here and new tools built using the API can be used to communicate complex data and introduce non-scientists to the concept of scenarios, thus successfully fulfilling the USGS mandate of sharing datasets and science with the public.
